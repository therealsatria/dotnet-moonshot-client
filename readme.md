- .NET SDK (latest version recommended)

### Installation
1. Clone the repository
```bash
git clone https://github.com/yourusername/moonshot-dotnet.git
cd moonshot-dotnet
```

2. Build the project
```bash
dotnet build
```

3. Run the project
```bash
dotnet run
```
